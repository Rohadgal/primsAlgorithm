# primsAlgorithm
Prim's algorith in C++ using an adjacency matrix

- For the code to work you must also download the Vector.hpp file that contains the vector class created for this implementation. 
Both Vector.hpp and Main.cpp must be in the same folder.
It is also possible not to download this file, but you would have to include the standard C++ std::vector library with an "#include <Vector>" and you would have to delete the include with the reference to the Vector.hpp file.
You should open the main.cpp file and compile and run that file in the editor of your choice.

- Para que el código funcione se debe de descargar también el archivo Vector.hpp que contienela clase vector creada para esta implementación. 
Tanto el archivo Vector.hpp y Main.cpp deben estar en la misma carpeta.
También es posible no descargar este archivo, pero se tendría que incluir la librería estandar de C++ std::vector con un "#include <Vector>" y se tendría que borrar el include con la referencia al archivo Vector.hpp
Se debe abrir el archivo main.cpp y compilar y correr ese archivo en el editor de sus preferencia.
